Instalación de MedSchedule v.0.7

1. Instalar "node modules" en CMD escribir "npm install"
2. En caso de problemas con cordova-plugins, escribir en CMD "ionic cap sync"
3. Para iniciar en Android Studio, en cmd "ionic cap open android"
